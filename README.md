# 🏥 Healthcare Management System

A full-stack healthcare platform that enables patients to book appointments, doctors to manage consultations, and admins to oversee the system.

## 🚀 Live Demo

https://healthcare-management-system-1-r0lk.onrender.com

## ✨ Features

* Patient Registration & Login
* Doctor Registration & Approval
* Appointment Booking System
* Razorpay Payment Integration
* Email Notifications (Brevo SMTP)
* Invoice Generation
* Video Consultation
* Admin Dashboard
* Doctor Dashboard
* Patient Dashboard

## 🛠️ Tech Stack

**Frontend:** React.js, Axios, React Router, Framer Motion

**Backend:** Node.js, Express.js, MongoDB Atlas, JWT

**Services:** Razorpay, Brevo SMTP, Cloudinary
